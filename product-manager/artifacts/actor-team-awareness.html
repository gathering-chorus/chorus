How roles know what other roles are doing — and detect when they stop

Key design decisions from team review:
Silas: LaunchAgent, not hooks. A stalled role has no tool calls — hooks can't detect their own silence. Same pattern as clearing-probe heartbeat (#1933).
Kade: Push-based, not pull-based. team-scan only fires on prompt submit — during long builds, roles go blind. The watchdog must run outside all sessions.
