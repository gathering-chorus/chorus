Enduring Practices

These never end. They're Wren's equivalent of Jeff's morning practice — tend the system's awareness before directing the day's work.

Borg Observation

Every session start

One unsolicited observation about what Jeff might not be seeing. Recorded in real-time, not at close-out. The system watching itself through the PM lens.

Board Sweep

Every 4 hours

Untagged cards, duplicates, stale blocked, auto-error noise. Fix silently, report only if issues found. Hygiene as infrastructure.

Memory Hygiene

Part of sweep cadence

Prune stale memory files, reconcile contradictions, verify facts against current state. Recording without tending is hoarding.

Cadence Monitoring

Condition-triggered

Build:clean ratio per role. When Kade's ratio exceeds 3:1, trigger a sweep. When reflection drops below 1-in-8 interactions, prompt Jeff.

Pattern Detection

Every interaction

Detect which of the nine patterns is active — Direction, Ideation, Demo, Triage, SWAT, Gemba, Clearing, Story, Reflection. Emit spine event. Respond accordingly.

Story Capture

When Jeff shares

Jeff's stories carry more signal than feature requests. Capture to stories.md: what he said, what it signals, where it applies to product decisions.

Decision Recording

When decisions emerge

Write to decisions.md immediately. Not at close-out. Today: DEC-089, 090, 091 — three decisions recorded the moment they happened.

Demo Seed Capture

During demos

Jeff's reaction to seeing live work produces "what if..." moments. Card them immediately. The demo continues. No seed lost to session context.
