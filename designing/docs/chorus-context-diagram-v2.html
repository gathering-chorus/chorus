Why Each Layer Exists

Shared Awareness
The product. Everything below exists so that Jeff, Wren, Silas, and Kade share one picture of what happened, what's happening, and what matters. History, sessions, stories, decisions, search, memory — indexed and queryable.
OWL/RDF — Semantic Layer
The meaning layer. Awareness collects signals; OWL/RDF says what they mean. Three ontologies (borg, chorus, jb) bridged by framework.ttl so one SPARQL query traces domain→service→gate→role. Without this, signals are noise. Prior art: US Patent 9,552,400 B2.
Loom
The team. Roles, the Clearing, interactions, briefs, cards, skills, policies, decisions, practices. How three AI roles and one human coordinate without Jeff becoming the message relay.
Protocol
The nervous system. Spine events and gates. Every action emits a signal; every transition is gated. The spine makes work visible; gates make quality automatic.
Framework
The contracts. ICDs define what data looks like before code touches it. Service designs define what services do before they're built. Domain pages show the current truth. BDD gates enforce it. Define, map, build.
Apps
The products Jeff uses. Gathering (personal knowledge), Chorus (team coordination), Borg (codebase analysis). Each consumes services, respects gates, and emits spine events.
Observability
The eyes. Alerts, logs, health checks, metrics. Roles see problems before Jeff does. Every service is monitored; every failure is an event, not a surprise.
Infrastructure
The ground. Two Mac minis (Library + Bedroom), LaunchAgents, Cloudflare tunnel, deploy tooling, codebase. Native services, no Docker. Everything above depends on this staying up.

Reading the Model

Vertical = dependency. Each layer depends on everything below it. If Infrastructure goes down, everything above it fails. If the Semantic Layer is wrong, Awareness reports garbage.

Horizontal = transaction. The seed journey shows how a signal flows from Jeff through every layer to become shared awareness. Quality gates check at every transition. Services provide the APIs that each layer consumes.

The dashed line matters. OWL/RDF is the only dashed layer — it's a horizontal capability that spans all layers, not a service you call. It's the schema that gives structure to everything above and below.
