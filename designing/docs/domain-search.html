Cross-cutting • Three search layers • Chorus value stream

The unified search surface across all of Gathering. Three layers working together: SQLite FTS5 for fast keyword search (154K items), LanceDB for semantic/vector search (15K docs), and Fuseki SPARQL for structured graph queries. Hybrid mode combines results. DEC-074 establishes the search hierarchy for roles: Chorus first, then codebase, then filesystem.
