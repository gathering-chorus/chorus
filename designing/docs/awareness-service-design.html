Shared Awareness — Service Design

2026-04-04 — Wren (driver) + Silas (navigator), pair on #2022. Source: Jeff's graph-paper sketch + deployment inventory.

Any role can answer "who is active, what are they doing, and when did they last emit?" without asking Jeff. When this breaks, Jeff becomes the monitoring system. That is the highest-cost failure mode.

Forcing Functions

Components that fire mechanically — no opt-in required.

Forcing FunctionTriggerWhat it does
Compound loopEvery UserPromptSubmitHybrid search + memory scan, inject context. Role doesn't choose to search.
Ops awarenessEvery PostToolUseCheck API health, hook server PID. Surface degradation or stay silent.
Team scanEvery UserPromptSubmitInject other roles' state and queued messages. No role is blind.
Alert runnerEvery 60 secondsCheck tunnel, daily review, hook server. Fire to Bridge AND terminal (#2037).
WatchdogEvery 60 secondsCheck role staleness. Flag if no state change in threshold (#2033 fixed today).
