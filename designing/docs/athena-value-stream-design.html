2 · athena-forge — the engine

athena-forge is owl-api renamed and re-homed as Athena's core engine. forge not cast: a cast is poured once and set; a forge is re-worked — the generator re-runs every time the OWL changes, re-forging the surfaces. Iterative craft.

  • Zero-dep Rust CLI binary (ADR-032 §1: std-only HTTP, SPARQL via curl); deployed signed like the other chorus-* binaries; live on :3360.
  • Today: generate (graph → routes.json) + serve (off the Node loop). Built #3350, ilities #3354/#3364.
  • The extension (the build target): generate grows from "routes.json" to routes + MCP + skill + page (the template's forge mechanism), plus static-HTML serving (the JX page).
  • Wren-owned (Jeff: "this is urs bc it is the core of athena"). Becomes the athena-* verb family + chorus_athena orchestrator.
