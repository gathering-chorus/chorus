## How It Works

```

  shared/                 # 10 fragments used by 2+ roles
  roles/silas/            # 13 Silas-specific fragments

messages/scripts/claudemd-gen.sh
  claudemd-gen.sh              # Generate all 3
  claudemd-gen.sh silas        # Generate yours only

  claudemd-gen.sh --check      # Exit 1 if stale
```
