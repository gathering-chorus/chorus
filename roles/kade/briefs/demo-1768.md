# Demo: #1768 — Sexuality Media Player

## What was built
Standalone Node.js app on Bedroom (port 8090) for browsing and playing sexuality photo sets and videos.

## AC Coverage
- [x] Three-panel left sidebar: playlist tree (top), photo picker (mid), video picker (bottom)
- [x] Photo picker: category, tag color, A-Z, text, and site filters; folders ≤ 125 photos; + button adds to playlist
- [x] Video picker: category, A-Z, text, site filters; + button on individual videos
- [x] Playlist: persistent tree (Photos → folders, Videos → files) saved as JSON on disk
- [x] Player surface: photos auto-play at configurable interval with back/pause/forward; videos play inline with native controls
- [x] Photos rendered at highest quality filling player space
- [ ] Video sound routed to Bedroom HomePod at 50% volume — not yet wired
- [x] Direct file serving for photos and videos (no proxy needed, runs on Bedroom)
- [x] Folder names parsed dynamically for category + site filters using canonical studio list
- [x] Runs on Bedroom as standalone server (port 8090, nodemon for dev)

## How to verify
1. Open http://192.168.86.242:8090
2. Photos: filter by category/tag/A-Z/text/site, expand model, click set to preview, + to add to playlist
3. Videos: same filters, expand model, click video to preview, + to add to playlist
4. Click playlist items to load into player
5. Photos: use Play/Prev/Next and speed slider for slideshow
6. Videos: native HTML5 controls

19/19 tests passing — folder name parsing, category extraction, tag detection
