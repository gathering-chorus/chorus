The Self hub. Contains: About (Jeff), People, Stories, Search, and a Self submenu with Value Stream, Domains (ontologies), Gathering Graph, Analytics (Jeff Dashboard, Voice, Attention), and Philosophy (Heidegger, Homeostasis, Self Portrait).

### 2. Value stream order
Gathering → Sowing → Growing → Practicing → Harvesting → Reflecting → Chorus → System. Not alphabetical, not arbitrary — this is the data flow.

- People: Gathering only (remove from Practicing and Harvesting)

ADRs (14), Architecture docs (11), Engineering docs (9), Chorus Process docs (10) — each should be ONE nav item pointing to a collapsible index page, not N individual nav items. The user clicks "ADRs" and gets a page where they can expand/collapse each one.

### 8. Reflecting slim
