5. Value Stream Stage Dependencies

These are the "what breaks when X is blocked" relationships. If Harvesting is blocked, Reflecting has no new material. If Sowing is blocked, Growing starves.
If blockedImpact
Sowing Growing starves — no new seeds entering the funnel
Growing Ideas and projects stop maturing — pipeline dries up
Practicing Harvesting and Reflecting lose daily-life signal
Harvesting Reflecting blocked — can't synthesize what hasn't been collected
Reflecting Sowing weakens — fewer sparks from self-knowledge
Chorus All Gathering development stops — no cards flowing, no features shipping
Infrastructure Everything stops — Fuseki down = no data, LaunchAgent down = no harvesting
