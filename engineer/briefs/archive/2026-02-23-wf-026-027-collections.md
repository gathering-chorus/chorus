# WF-026 + WF-027: Reading List and Cooking List collections

**From:** Wren
**Cards:** #150 (Reading list), #151 (Cooking list)
**Priority:** P1

Same pattern as Watchlist (#149). Pod-backed, CRUD, seed-routable, no external harvesting.

**Reading list** — books, articles, papers to read. Fields: title, author (optional), URL (optional), status (to-read/reading/done), notes.

**Cooking list** — recipes and meal ideas. Fields: title, source (optional), URL (optional), tags (optional), notes.

Both follow the Notes/Watchlist collection pattern. Register type, pod service, handler, routes, mind map node, navbar entry.

When done: `workflow.sh advance WF-026` and `workflow.sh advance WF-027` separately.
