Ambient Hooks (fire across every step)

  • SessionStart — context envelope injection.
  • chorus-hook-shim — Rust binary at ~/.chorus/bin/; wraps every Edit / Write / Bash PreToolUse and PostToolUse.
  • write_scrubber — blocks credentials in writes.
  • canonical_write_guard — blocks role edits under canonical when CHORUS_WERK_ENABLE=1; redirects to $<ROLE>_WERK.
  • icd_write_gate — blocks harvester writes without an ICD provider (DEC-095).
  • chrome_tab_gate — blocks raw open <url>; routes via chrome-window.sh.
  • clock_sync — wall-clock validation.
  • pair_enforcement, memory_gate, observer, autonomy_guard, e2e_responder, interaction_pattern — additional behavioral hooks at PreToolUse / PostToolUse boundaries.
  • git-queue.sh branch-check (#2580); pre-commit (platform/hooks/pre-commit); pre-push (#2598/#2625/#2639).
  • MCP layer (chorus_pull_card, chorus_acp, chorus_nudge_message): typed refusal taxonomy replaces prose for those flows.

What is still PROSE-only (not substrate-enforced)

  • Step 5 ACK contract — recipient MUST reply substantively; sender re-nudges; escalation to Jeff after 2 retries. #2872 will make it substrate.
  • Step 6 show-don't-tell behavior — model self-policed.

Three former PROSE rows moved to HOOK under #2893: Step 2 chain-orchestration (mistagged-chore caught), Step 3 AC happy-path (derived checks), Step 4 stakes-brief lint ("Why this matters" required, mechanics-first refused).

What's enforced that isn't obvious from SKILL.md

  • Pre-flight is substrate, not model-grep. demo_preflight.rs → preflight.sh fires on Skill(demo) PreToolUse — you cannot run /demo with unchecked AC; the hook refuses before SKILL.md is read.
  • Show-gate's load-bearing fire is at /acp time, not /demo time. The PostToolUse-after-/demo registration fires too early; accept_gate.rs runs show-gate on Skill(acp) PreToolUse, reads Loki for the demo chain (started + Jeff-input window + preflight-passed), and refuses /acp if any precondition is missing.
  • "Jeff watched" is provable. jeff.input.delivered spine events in the window after card.demo.started are the evidence — not just that the agent posted a preflight comment. #2864 raised the bar from comment-shape to Jeff-attention-shape.
  • Self-accept refusal is in the hook. accept_gate.rs refuses builder-self-accept on code cards; Jeff override per DEC-048.
