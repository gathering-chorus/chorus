Gathering Value Stream & Domain Map

From Wren (PM) — Updated 2026-02-26 — Foundational product doc

Gathering's value isn't in storing data. It's in the movement from spark to self-understanding. The stream is a cycle — reflection generates new sparks.

The Value Stream — Short Loop

The moment-to-moment PDCA cycle. Minutes to hours. A spark arises, gets captured, sorted, placed, revisited, transformed, connected, reflected — and the reflection generates the next spark.

Mind Map Nodes — Long Loop

The coherence and learning cycle. Days to seasons. The mind map organizes life domains across an agricultural metaphor — sowing, growing, practicing, harvesting, gathering, reflecting. This is where planning, pattern recognition, and life architecture happen.

Mind Map Node Subtitle Value Stream Stages The Act
Sowing capture & seeds Spark, Capture Planting — raw material enters the system
Growing ideas & growth Triage, Settle Tending — sorting, placing, giving things a home
Harvesting collections Revisit, Transform Collecting — returning to what's matured
Gathering find & discover Connect Pulling together — cross-collection search
Reflecting inner world Reflect Understanding — patterns, meaning, self-knowledge
About who & what — Identity — Jeff's profile and product context
Admin tools & settings — Operations — system management
Why this naming: "Gathering" as a node was confusing — same word as the app. Renaming capture to Sowing and search to Gathering makes the metaphor coherent: Sow → Cultivate → Harvest → Gather → Reflect. The app name covers the whole cycle; no single node claims it.
